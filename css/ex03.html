CSS 선택자 - 전체 선택자, 태그 선택자

CSS 선택자는 HTML 요소를 선택하여 스타일을 적용하는 데 사용됩니다. 이 문서에서는 전체 선택자와 태그 선택자에 대해 설명합니다.

1. 전체 선택자 (Universal Selector)

전체 선택자는 모든 HTML 요소를 선택하는 데 사용됩니다. 별표(*) 기호로 표시됩니다. 전체 선택자를 사용하면 페이지의 모든 요소에 동일한 스타일을 적용할 수 있습니다.

위의 스타일은 모든 요소에 10픽셀의 여백과 패딩을 적용하고, 박스 크기 계산 방식을 변경합니다.

2. 태그 선택자 (Type Selector)

태그 선택자는 특정 HTML 태그를 선택하는 데 사용됩니다. 예를 들어, 모든 <p> 태그를 선택하려면 다음과 같이 작성합니다.

위의 스타일은 모든 단락(<p>) 요소의 글자 색상을 파란색으로 설정하고 글자 크기를 16픽셀로 지정합니다.
