CSS 적용하는 방법 3가지

웹 페이지에 CSS를 적용하는 세 가지 주요 방법이 있습니다:

1. 인라인 스타일 (Inline Style)-기본

HTML 요소의 style 속성을 사용하여 직접 스타일을 지정하는 방법입니다. 태그 안에 style태그를 넣어서 스타일링.

이 문장은 인라인 스타일로 빨간색과 굵은 글씨로 표시됩니다.

2. 내부 스타일 시트 (Internal Style Sheet)-인스턴스

HTML 문서의 <head> 섹션 내에 <style> 태그를 사용하여 스타일을 정의하는 방법입니다.

이 문장은 내부 스타일 시트로 녹색과 20픽셀 크기로 표시됩니다.

3. 외부 스타일 시트 (External Style Sheet)-대용량

별도의 CSS 파일을 만들어 HTML 문서에 링크하는 방법입니다. 이 방법은 여러 페이지에서 동일한 스타일을 재사용할 수 있어 효율적입니다.

<link rel="stylesheet" href="styles.css" > 라고 선언.

이 문장은 외부 스타일 시트로 스타일링됩니다.
